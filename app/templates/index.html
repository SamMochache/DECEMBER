Finance Tracker: Smart & Secure Expense Management

Finance Tracker is a streamlined expense tracking system that securely syncs financial transactions via Plaid API and allows users to categorize expenses for precise budgeting.
A real-time analytics dashboard provides clear insights into spending patterns, while Google OAuth authentication ensures secure access. Designed for ease of use and financial clarity, Finance Tracker simplifies personal finance management.

Vulnerability Management System (VMS)

The Vulnerability Management System (VMS) is a Django-based cybersecurity tool that helps security teams identify, track, and manage vulnerabilities across network infrastructures.
It integrates Nmap for automated scanning and VirusTotal for threat intelligence, providing a comprehensive security assessment. Key features include role-based access control, secure API authentication, logging & auditing, and a dashboard for vulnerability reports. Designed for cloud deployment, VMS ensures proactive security management with automation and real-time insights.
